1. **Claude Mem** — [github.com/thedotmack/claude-mem](https://github.com/thedotmack/claude-mem)
   Claude Code 的结构化记忆系统。让 Agent 跨会话记住上下文，不再每次从零开始。

2. **UI UX Pro Max** — [github.com/nextlevelbuild](https://github.com/nextlevelbuild)
   前端设计 Skill，让 Claude Code 产出专业级 UI 而不是千篇一律的 AI 风格。

3. **n8n-MCP** — [github.com/czlonkowski/n8n-mcp](https://github.com/czlonkowski/n8n-mcp)
   将 n8n 工作流自动化能力接入 Claude Code，构建自动化管道。

4. **Obsidian Skills** — [github.com/kepano/obsidian-skills](https://github.com/kepano/obsidian-skills)
   让 Claude Code 与 Obsidian 笔记深度集成的 Skill 集。

5. **LightRAG** — [github.com/hkuds/lightrag](https://github.com/hkuds/lightrag)
   基于图的检索增强生成。彻底改变了 Claude 处理长上下文任务的方式。

6. **Everything Claude Code** — [github.com/affaan-m/everything-claude-code](https://github.com/affaan-m/everything-claude-code)
   Claude Code 资源大全，包含教程、配置和最佳实践。

7. **Superpowers** — [github.com/obra/superpowers](https://github.com/obra/superpowers)
   Claude Code 最受欢迎的 Skill 集之一。自动化头脑风暴→规格→计划→执行→审查→发布的完整工作流。

8. **Awesome Claude Code** — [github.com/hesreallyhim/awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)
   Claude Code 精选资源列表，工具、Skill、MCP 服务器一网打尽。

9. **GSD (Get Shit Done)** — [github.com/gsd-build/get-shit-done](https://github.com/gsd-build/get-shit-done)
   专注执行的 Claude Code 框架，让 Agent 快速交付实际成果。

---

> Claude Code 正在成为一个平台，而不仅仅是一个工具。6 个月前它只是一个写代码的终端。现在：插件市场、Skill 系统、MCP 集成、持久记忆、数千星的社区仓库。这正是 VS Code 赢的方式——不是编辑器，而是生态系统。
